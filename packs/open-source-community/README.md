# Open-Source Community Pack

For community-governed open-source projects. Public-by-default, RFC-governance, sunset-aware.

## What you get

- `profiles/external-comms.yaml` — community review windows (no MPA clock; community-RFC-period instead), no funding-acknowledgement requirement, contributor recognition.
- `profiles/phase-gate.yaml` — Incubation → Active → Maintained → Archived lifecycle.
- `reporting-matrix-defaults.yaml` — release-note posts, contributor digest, RFC pipeline updates.
